# Bedrock Client Addons
Pure client side packs, meaning works as 'texture pack' even in multiplayer.

# Chronos

- Turns day counter into a working calendar to track time over large scales on HUD
- Adds 12 Hour clock on HUD scaled to 5 minute intervals

# Personalized Villagers

- Randomizes Villager textures based on given name

# Horse Stats

- Speed: After running measures speed in blocks/meters per second
- Jump: Jumping once grades the jump power 
- Health: Displays health more accurately than vanilla heart icons
- Rankings: Ranks horse stats which could help with breeding.
